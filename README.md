# Authentication REST API

## Features
> 1. Register a user.
> 2. Verify user's email address.
> 3. Send forgot password reset email.
> 4. Reset password
> 5. Get current user.
> 6. Login.
> 7. Access tokens.
> 8. Refresh tokens.

## Tech used
> 1. Node.JS
> 2. Typescript
> 3. Typegoose
> 4. Zod
> 5. JWT
